## Introduction to Git
### Intialize
```sh
git init
```
### Commit
```sh
git add -A && git commit -m "..."
```
### Push
```sh
git push 
```
### Others
```sh
git checkout master
git checkout <hash>
git checkout -b <branch_name>
```
